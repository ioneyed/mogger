Mogger API Documentation

Mogger is a multi-server blogging platform used to teach various languages as well as show that all languages can be used as a web service. Each mogger-backend has to follow this API to work within the mogger ecosystem.

Application Mime-types application/vnd.mogger.post+json;v={version} application/vnd.mogger.post+xml;v={version}

    "code":200,
    "message":"success"

Example:

        "comments":{"href":"/api/{category}/posts/{pid}/comments"},
        "related":{"href":"/api/{category}/posts/{pid}/related"},
        "like":{"href":"/api/{category}/posts/{pid}/like"}
      }
    }]
  }
}

    "pagination":{
      "next": {"href":"/api/{:category}/posts/{:pid}/next"},

  "modified":"2014-11-20 22:00:33.511-05",

      "title":"Part 2",
      "href": "/api/{:category}/posts/{:pid}"
